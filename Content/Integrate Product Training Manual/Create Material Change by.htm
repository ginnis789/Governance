Exercise 4 - Create Material Change by Importing a Recording

A BDC script can be uploaded, if the script has been created in SAP, using the SHDB transaction. It is preferable to upload a script when:

  • A script was already recorded in SAP using the SHDB transaction (as there is no need to re-record it).
  • A browser other than Internet Explorer is used.
  • The client security policy restricts recording.

Go to the SHDB transaction in SAP and perform the BDC recording directly in SAP. Save the recording as a flat file, and then upload and import it into a template.

Create an SHDB recording to change the Division (SPART) from 09 to 10 for all materials in Material Group (MATKL) 020.

The following steps are required:

  • Create a SHDB recording in SAP.
  • Create a template and import the recording.
  • Add a Process to TrainingXX Category.
  • Use txChangeDivision_JDEInt view in the sdbNorthwindXX database.
  • Add a Where Clause.
  • Post the changes.
