1.1 Create a BDC Script

When recording a BDC script, SAP launches in learning or listening mode, which records the user's actions in SAP. Once the recording is complete, the user exits the transaction in SAP, and the recording is imported into Integrate.

NOTE:Internet Explorer MUST be used to record from within the DSP platform.

Create a BDC script to post legacy materials into SAP.

To create a template:

  1. Locate TrainingXX Category.
  1. Click Templates .
  1. Enter XXMaterials_JDE in Template Name field.

NOTE:The Template Name must be unique. The template name can contain A-Z, 0-9, and underscore. No special characters are allowed in template names, and Integrate will replace them with underscores when the template is saved.

  1. Enter JDE Materials in Description field.
  1. Select BDC Script from Type list box.
  1. Select SAP from SAP Connection ID list box.

NOTE:The SAP Connection ID defines which SAP system is used to gather metadata for the template. Type defines the type of SAP interaction that the template will process.

  1. Click the Save button, the Vertical View displays.
  1. Enter MM01 in Transaction Code field.
  1. Click the Save button.
  1. Click Record , a message displays.
  1. Click the OK button.

NOTE:SAP opens in learning mode using the defined transaction code, which uses the same recording mechanism as the SHDB transaction in SAP to track the fields modified by a user during the recording. The recording overwrites all template data with data from the recording.

NOTE:Some actions performed in Integrate require that a user enter an SAP user name and password. In general, a user enters these credentials once per session, and the platform retains these values and reuses them throughout the session. If a session times out, or if a user logs out, the platform does not Save the credentials. Integrate parameters can control whether a user must enter credentials for the SAP system every time these actions are performed or only the initial time the actions are performed during a session.

  1. Enter trainingXX in SAP User ID field.
  1. Enter trainingXX in SAP Password field.
  1. Click the Save button.

NOTE:An SAP GUI Security pop-up box may display. If so, select Always Allow, and click OK. The recording must be re-recorded if this message displays.

  1. Enter Finished Product in Material Type field.
