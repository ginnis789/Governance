2.0 Self-Directed – Add a Process to the Category

To create a Process in TrainingXX.

  1. Enter 20 in Priority field.
  2. Name the Process XXSemiFinished_JDE.
  3. Enter Semi Finished Materials from JDE in Description field.
  4. Select the XXMaterial_JDE Template from the Template ID list box.
  5. Select sdbNorthwindXX as the Data Source ID.
  6. Select txSemiFinishedMaterials_JDEInt from the View Name list box.
  7. Click the Save button.
