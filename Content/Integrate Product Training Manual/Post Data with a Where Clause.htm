3.2 Post Data with a Where Clause

A Where Clause can be added to Process Posts to restrict the data that is posted. Examples can be large sets of data restricted in Batches, or conditional situations. When posting a BDC, GUI or BAPI/RFC template with more than one loop, the WHERE CLAUSE must apply to all views mapped to the loops.

Add a Where Clause to the process post to only update Weight Unit for materials that are in External Material Group (EXTWG) 01.

To add a Where Clause:

  1. Locate XXChangeWeightUnit Process.
  2. Click Post , the Process Post displays in the Dependent pane.
  3. Click the Vertical View .
  4. Click View Primary Loop Data to view the filtered data in the view.

NOTE:25 records display.

  1. Click the Browser Back button.
  2. Click Edit button.
  3. Enter EXTWG = ‘01’ in Where Clause field.
  4. Click the Save button.
  5. Click View Primary Loop Data to view the filtered data that will be posted.

NOTE:If there is an error in the Where Clause an error displays when the Primary Loop Data icon is clicked.

  1. Click the Browser Back button.
  2. Close the Vertical View.
  3. Click Post , a message displays.
  4. Click the OK button.
  5. Enter the SAP credentials if prompted.

NOTE:A message displays indicating 10 records were successful.

  1. Click View Messages to view the SAP returned messages.
  2. Click the Browser Back button.
