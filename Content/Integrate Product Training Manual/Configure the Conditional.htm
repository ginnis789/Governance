9.1 Configure the Conditional

Configure the Conditional to not load the Basic Material if the Value is CR25.

To configure a conditional:

  1. Click Categories on the Navigation pane.
  2. Click Templates for TrainingXX.
  3. Locate the XXNewMaterials Template.
  4. Click Activate/Deactivate to activate the Template, a message displays.
  5. Click the OK button.

NOTE:In order to determine the fields on the BDC recording, the Template must be activated to retrieve the Meta Data. It must be deactivated to add the Conditional.

  1. Click Activate/Deactivate to deactivate the Template, a message displays.
  2. Click the OK button.
  3. Click Vertical View for the Template.
  4. Click Configuration tab.
  5. Click BDC Screen .

NOTE:When recording for a Conditional, navigate to each screen and then back to the main screen.

  1. Click Vertical View for Priority 40, screen 4004 – SAPLMGMM.
  2. Click the Conditionals .
  3. Select MARA-WRKST from the Conditional Field Name list box.
  4. Select Equals from the Conditional Type list box.
  5. Enter Metal in the Conditional Value text box.
  6. Click the Save button.
