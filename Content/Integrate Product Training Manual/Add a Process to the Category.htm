7.2 Add a Process to the Category and Configure Loop

Add the XXGLAccounts Template as a Process to TrainingXX.

To add the process:

  1. Click Processes .
  2. Click the Add button.
  3. Enter 80 in Priority field.
  4. Enter XXGLAccounts in Name field.
  5. Enter GL Account Maintenance in Description field.
  6. Select XXGLAccounts from Template ID list box.
  7. Select sdbNorthwindXX from Data Source ID list box.

NOTE:Do not select a view.

  1. Click the Save button.
  2. Close the Vertical View.
  3. Locate XXGLAccounts process.
  4. Click Templates .
  5. Click Loops .
  6. Click the Auto Generate Database Objects , a message displays.
  7. Click the OK button.

NOTE:The View Name has been auto generated and the Primary Key Column Name is populated.

NOTE:The View has the Header information and the Component relationship. When the common columns have the same field name, Integrate automatically determines the relationship between the two. It is very common that the relationship will not be completely obvious to Integrate. Click the Relationship button on the child template to manually choose the relationship to the child, and what should be looped. The Header ID is 1, and linked to a Parent ID column to the child template. Aligning the data for each parent will run all the children. For example, line the Customer order and order detail. It is similar to how a join works in a view forming the relationship from the Order to the Order Detail.
