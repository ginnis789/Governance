5.0 Copy a Template

Existing templates can be copied to create a new template in any category. A copy of a template includes all settings in the original template, including:

  • Screens, fields, loops and conditionals in the case of BDC Scripts and GUI Scripts.
  • Custom commands in the case of GUI Scripts.
  • Structures and fields in the case of User Defined, Batch Load, Direct Load and IDOC templates.

NOTE:Copy template functionality is not available for BAPI/RFC template types.

A copy of a template does not copy the processes associated with the original template.

NOTE:A template can be copied whether the template is active or not.

To copy a template:

  1. Click Categories on the Navigation pane.
  2. Locate the Material Master Category.
  3. Click Templates .
  4. Locate the ExtendXX Template.
  5. Click Vertical View .
  6. Click Copy tab.
  7. Click Copy Template .
  8. Click Edit.

  9. Select TrainingXX from Copy To Category list box.
  10. Enter XXExtendToPlant in Copy To Template Name field.
  11. Enter Extend Materials in Copy To Template Description field.
  12. Click the Save button.
  13. Click Execute , a message displays.
  14. Click OK.
  15. Close the Vertical View.
  16. Click Categories on the Navigation pane.
  17. Locate the TrainingXX Category.
  1. Locate XXExtendToPlant template.
  2. Click Vertical View .
  3. Click Configuration tab.
  4. Click BDC Screen to view the imported recording.
  5. Click the Browser Back button.
  6. Close the Vertical View.

NOTE:The template must be activated before it can be added to a Process.

  1. Click Activate/Deactivate to activate the template.
  2. Click the OK button.
