10.0 Create a Template Using a BAPI

Configuring a BAPI template includes selecting the name of the Function Module in SAP. Integrate then pulls the definition of the Function Module from SAP to complete the template.

Before creating a template, ensure the RFCs and BAPIs for the SAP connection have been downloaded.

Create a Template using the BAPI_BANK_GETLIST to get a list of Bank data.

To create a Template:

  1. Click Add on the Template page for TrainingXX Category.
  2. Enter XXGetBankList in Template Name field.
  3. Enter Bank List BAPI in Description field.
  4. Select BAPI/RFC from Type list box.
  5. Select SAP from SAP Connection ID list box.
  6. Click the Save button, the Vertical View displays.
  7. Enter BAPI_BANK_GETLIST in Function Name list box.
