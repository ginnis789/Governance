1.4 View Messages

To view the messages:

  1. Click View Messages on the Process Post page.

NOTE:The top material number and the last material number posted in the Message column. For example, in the screen shot below the top number is 2060 and the last number is 2068. A record displays for each record in the dataset, with a message returned from SAP. Data on the Messages page is used to verify that data posted correctly in SAP. Click the Vertical View to view the Return Value(s). The Message table is not automatically cleared; it must be manually cleaned out. To delete the messages, go to the Vertical View of the Process Post and click Delete Messages .
