10.1 Create a Process

Create a process to get data using the BAPI Template.

To create a Process:

  1. Click Processes .
  1. Click the Add button.
  1. Enter 70 in Priority field.
  1. Enter XXGetBankList in Name field.
  1. Enter BAPI Bank List in Description field.
  1. Select XXGetBankList from Template ID list box.
  1. Select sdbNorthwindXX from Data Source ID list box.

NOTE:Do not select a view.

  1. Click the Save button.
  1. Close Vertical View.
  1. Click Templates for XXGetBankList Process.
  1. Click Loops in Dependent pane.

NOTE:Integrate adds a process template loop for each loop in the BAPI Definition. The Loop Name for each loop should match the loop names on the BAPI/RFC Definitions page. The loop contains the fields in the definition. The Loops and the fields associated with the Loops display in a split screen.
