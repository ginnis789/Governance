5.1 Create a Process with Multiple Templates

Create a process and add the XXMaterials_JDE and the copied XXExtend template.

To create a process with multiple templates:

  1. Click the Browser Back button.
  2. Click Processes .
  3. Click the Add button.
  4. Enter 50 in Priority field.
  5. Enter XXCreateAndExtend in Name field.
  6. Enter Create and Extend Materials in Description field.
  7. Select XXMaterial_JDE from Template ID list box.
  8. Select sdbNorthwindXX from Data Source ID list box.
  9. Select txStringingFinishedMaterialsInt from View Name list box.
  10. Click Advanced Properties tab.
  11. Select Synchronous from Post Execution Method list box.
  12. Enter 2 in Delay Between Templates field.
  13. Enter 250 in Delay Between Records field.

NOTE:A process that uses the Asynchronous post execution method, the option that is selected by default, executes the post for each template sequentially without waiting for the template post to return before executing the next template post. Asynchronous posting may result in locking, especially when posting a combination of GUI and BDC templates in the same process. A process that uses the Synchronous post execution method executes the post for each template, but waits for the post to return before executing the next post. Synchronous is a slower execution method but prevents most recording locking situations and is necessary for processes with multiple templates.

  1. Click the Save button.
  2. Close the Vertical View.
  3. Locate XXCreateAndExtend Process.
  4. Click Templates , the template displays in the Dependent pane.
  5. Click the Add button in the Dependent pane.
  6. Enter 20 in the Priority field.

NOTE:Set the priority order of the templates in the order they should be processed using transaction stringing. Templates are processed in ascending priority order. A template with a low priority is processed before a template with a high priority.
