4.2 Self-Directed – Create a Process and Post

Create a process and add a Where Clause to change the Division (SPART) from 09 to 10 for all materials in Material Group (MATKL) 020.

The following steps are required:

  • Enter 40 in the Priority field.
  • Name the Process XXChangeDivision.
  • Enter Change Material Division in the Description field.
  • Select the XXChangeDivisionTemplate.
  • Use the txChangeDivision_JDEInt view in the sdbNorthwindXX database.
  • Select Loop Primary Key, Old Material (BISMT).
  • Add the Where Clause.
  • Post the changes.
  • View the SAP returned messages.
