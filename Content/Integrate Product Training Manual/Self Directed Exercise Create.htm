3.1 Self-Directed Exercise – Create a Process

Create a process in TrainingXX Category for the new Template.

  1. Enter 30 in Priority field.
  2. Name the Process XXChangeWeightUnit.
  3. Enter Change Weight Unit for Ext Mat 01 in Description field.
  4. Select the XXChangeWeightUnit Template.
  5. Select sdbNorthwindXX as the Data Source ID.
  6. Select txChangeWeightUnitInt as the View Name.
  7. Select BISMT as the Primary Key for the Loops.
  8. Activate the process.
