2.1 Configure Field Mappings

If the fields in the view are named with valid SAP fields, Integrate completes the mapping automatically based on the fields in the recording. If it cannot recognize the field names, the fields have to be mapped manually.

Process template loop field mappings provide the link between a field on a template and on a database view. Each unique field on the template can be mapped to either a field in the database view or assigned to a fixed value. If there is no mapping or fixed value, then the fields are not mapped and will be ignored in the posting.

Map the fields in the txSemiFinishedMaterials_JDEInt view to the correct SAP fields.

To map the fields:

  1. Click to close the Vertical View.
  2. Locate XXSemiFinished_JDE.
  3. Click Templates for the Process.
  4. Click Loops for the Process Template in the Dependent pane.
  5. Click the Edit button in the Parent pane.
  6. Select Old Material Number from Primary Key Column Name list box.
  7. Click the Save button.

NOTE:The Process Template Loop Field Mappings page displays either to the left or at the bottom of the screen. To change the location, click the reposition icon to change the position of the field mappings. All the fields associated with the template recording display in Template Field Unique Name column.
