4.0 Create a SHDB Recording

Create a SHDB recording to change the Division based on the Material Group.

To create a SHDB recording:

  1. Log into SAP.
  1. Enter SHDB in Command field.
  2. Click New Recording button.
  3. Enter XXDivision in Recording field.
  4. Enter MM02 in Transaction Code field.
  5. Click the Start recording button.
  6. Enter the First Number from the messages in Exercise 1.4 in Material field.
  7. Click .
  8. Select Basic Data 1.
  9. Click .

NOTE:The SHDB recording will capture all fields that contain data on the screen.

  1. Click Save .
  2. Click Export .

  3. Name the file XXChangeDivision.
  4. Save the file to the desktop.
