Overview

Integrate™ is a component of the Data Stewardship Platform™ (DSP) that contains loading mechanisms to push data into one or many business systems.

If the Business System is an SAP ERP system, Integrate comes pre-configured to load using the following SAP proprietary load mechanisms:

  • Batch Data Communication (BDC) processing
  • Graphical User Interface (GUI) scripting
  • A Remote Function Call (RFC) or a BAPI

Integrate can also create and transfer user defined text files formatted as Delimited, Fixed Width or XML.

Other types of interfacing technologies include the execution of SAP Data Services Jobs, as well as a Custom Template API allowing the creation and execution of custom integrations.

Integrate can be used by any platform component (including the framework), to post data to the target system.

Integrate is organized into templates and processes. A template is based on a template type and defines how data is posted into a business system. Templates are not tied to data, but rather act as an independent guide for posting that can be assigned to one or more processes. A process is a series of posting steps that defines how Integrate takes data from the component and loads it into the systems.

NOTE:Integrate is an automation tool that allows users to leverage data loading capabilities provided by external systems. Knowledge of how to use these systems loading technologies is required to use Integrate; the documentation and technical support included does not cover specific usage details or how these technologies work. The exercises provided are examples only, and do not supply comprehensive instructions to suit all situations.

Use Integrate to:

  • Manage Templates
  • Manage Processes
  • Monitor Posts
  • Post Data with a BDC Script
  • Post Data with a GUI Script
  • Post Data Using a BAPI
  • Post Data Using RFC Execution
  • Create and Transfer User Defined Flat Files
  • Create and Transfer XML Files
  • Post a Process with Multiple Templates
  • Create a Custom Posting
  • Register and Execute After Post Rules
  • Troubleshoot Posting Issues

Who should use this manual?

The Integrate Training Manual is created to train individuals responsible for the setup and execution of Export and/or Loading portions of DSP, and DSP Solution Implementations.

General Notes

  • Each student is assigned a DSP Training ID and a password.
  • Each student is assigned an SAP User ID and password as part of the exercises.
  • Each student is asked to create templates and processes using specified object names prefixed with XX, which references the assigned Training ID number.
