Exercise 9 - Add a Conditional to a Template

A conditional is a data condition that, if true, will result in the inclusion of the data from the process template field mappings associated with the screen in the resulting posting script. If the data condition is false, the screen will be left out of the posting script.

Each conditional is data record specific. For example, if 10 records are posted, Integrate evaluates the conditional for each record and applies it to the data in the record only.

If a conditional is not applied to a screen, the screen will be included in the posting script.

Only one conditional can be configured per BDC screen.

NOTE:Conditionals cannot be modified when a template is active. Conditionals can be applied to custom fields.
