7.0 Create a Template

To Create a GL Account template:

  1. Navigate to Templates.
  2. Click the Add button.
  3. Enter XXGLAccounts in Template Name field.
  4. Enter GL Accounts Loop in the Description field.
  5. Select BDC Script from Type list box.
  6. Select SAP from SAP Connection ID list box.
  7. Click the Save button, the Vertical View displays.
  8. Enter FB50 in Transaction Code list box.
  9. Click the Save button.
  10. Click to expand the File Import section.
  11. Click the Upload a File .
  12. Select ImportBDC_FB50 on the desktop.
  13. Click the Import once the file has uploaded.
  14. Click the OK button.
