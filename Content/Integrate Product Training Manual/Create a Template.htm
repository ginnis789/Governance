4.1 Create a Template

Create a new template and import the SHDB recording.

To create a template:

  1. Return to Integrate.
  2. Click Categories on the Navigation pane.
  3. Locate TrainingXX Category.
  4. Click Templates .
  5. Click the Add button.
  6. Enter XXChangeDivision in Template Name field.
  7. Enter Change Material Division in Description field.
  8. Select BDC Script from Type list box.
  1. Click the Save button, the Vertical View displays.
  2. Enter MM02 in Transaction Code field.
  3. Click the Save button.
  4. Click File Import line to expand the section.
  5. Click Upload , a window displays.
  6. Select XXChangeDivision file.
  7. Click the Open button.
  8. Click Import to import the data, a message displays.
  9. Click the OK button.
  10. Click Configuration tab.
  11. Click BDC Screen to view the imported recording.

NOTE:The SHDB recording capture all fields on the screen.

  1. Click the Browser Back button.
  2. Close the Vertical View.
  3. Locate XXChangeDivision template.
  4. Click Activate/Deactivate to activate the template, a message displays.
  5. Click the OK button.
