6.2 Run SM35 Session

The template created in Exercise 6.0 is missing a field. A SM35 session was completed to determine what field needs to be added to the view, and what SAP screen the view is on.

To determine the template issue:

  1. Log into the SAP GUI.
  2. Enter Transaction code SM35.

NOTE:SM35 is a session overview screen for Batch Input data transfers. Batch input is one of the primary ways in which data can be transferred into SAP. SM35 allows users to:

  • Process input sessions either in the foreground or background.
  • Check input sessions in the event that batch input sessions are processed on a schedule.
  • Analyze input errors. Input sessions offer detailed logging during the process.
  • Correcting input errors during foreground execution. Users can interactively correct data and re-process transactions.
  1. Locate TRAINXX in Session name column.
  2. Highlight TRAINXX row.

NOTE:To highlight a record in SM35, click on the blue box to the left of the record.

Session Overview in SAP: SM35

  1. Click the Process button on the SAP toolbar.
  2. Click the Display Errors Only radio button.
  3. Click the Process button.

NOTE:A message displays where the error occurs, and Tax Classifications field displays in yellow indicating a required field.

SAP Error in SM35

NOTE:To determine the technical information for the field and what view it sits on, the Technical Information needs to be viewed.

  1. Place the cursor in the field.
  2. Click F1 on the keyboard, the Performance Assistant window displays.
  3. Click Technical Information on SAP toolbar.

NOTE:The Tax field is on screen 4200 and located in table MG03STEUER.
